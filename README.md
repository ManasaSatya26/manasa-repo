# Lakshmi Prasanna — Classical Dance Instructor Portfolio

A single-page, fully responsive portfolio website for Lakshmi Prasanna, a certified Kuchipudi dancer and dance instructor applying for school teaching positions.

## Tech Stack

- **Next.js 15** (App Router) + React + TypeScript
- **Tailwind CSS** for styling
- **Framer Motion** for scroll-reveal and hover animations
- **Lucide React** for icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build & Deploy

```bash
npm run build
npm start
```

Deploy to [Vercel](https://vercel.com) with one click — no additional configuration required.

## Customization

- Replace placeholder contact info in `src/components/Contact.tsx`
- Swap gallery/certificate images in `src/components/Gallery.tsx` and `src/components/Certificates.tsx`
- Update the hero background image in `src/components/Hero.tsx`
